# Form Elements

This project uses [Formik](https://jaredpalmer.com/formik/) for forms by Jared Palmer.
This is an example form:

```jsx
<Formik
  initialValues={{
    name: '',
  }}
  onSubmit={values => {
    // same shape as initial values
    console.log(values)
  }}>
  {({ isValid, handleSubmit }) => {
    return (
      <Form>
        <Field name="name" />
        <Button
          ml="auto"
          mr="0px"
          onClick={() => handleSubmit()}
          disabled={!isValid}>
          submit
        </Button>
      </Form>
    )
  }}
</Formik>
```
